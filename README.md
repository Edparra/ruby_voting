# ruby_voting

Three workers are picked in the running for MVP. Most Valuable Programmer.
Show them it should be you by writing a program that counts all the votes and finds the winner.

Program should:
  *   Greet user and introduce 3 candidates
  *   User should be able to input 10 votes
     *  Take into account invalid votes
  *   Display the number of votes for each person and invalid votes
  *   Announce the winner
      *   Announce a tie only if there is a tie for first place
      
## Bonus
 
 Make the input hidden to keep the votes anonymous <br>
   **HINT** [Ruby Docs](http://www.ruby-doc.org/stdlib-2.0/libdoc/io/console/rdoc/IO.html#method-i-noecho)
